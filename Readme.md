
# canonical

  Component: returns the canonical URL for the page.

## Installation

    $ component install segmentio/canonical

## Example

```html
<html>
<head>
  <link rel="canonical" href="http://exampe.com" />
</head>
</html>
```

```js
var canonical = require('canonical');
canonical(); // http://example.com
```
